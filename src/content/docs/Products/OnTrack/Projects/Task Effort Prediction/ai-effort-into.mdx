## Executive Summary

This feature project aims to develop a system, including a machine learning
model for predicting the amount of effort required for students to complete
academic tasks. By analysing task metadata and extracting key information from
task sheets, the system will generate consistent, data-driven workload
estimates for students based on metrics derived from the OnTrack learning
management system.
